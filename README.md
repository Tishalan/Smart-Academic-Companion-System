# Smart Academic Companion System

A comprehensive full-stack academic management system with machine learning predictions for student success.

## Features

- **Role-based Access**: Separate dashboards for Admin, Lecturers, and Students
- **ML Predictions**: Dropout risk, graduation probability, and next module performance
- **Real-time Analytics**: Interactive charts and visualizations
- **Automated Alerts**: Risk detection and attendance warnings
- **PDF Reports**: Generate detailed reports
- **Responsive Design**: Works on desktop and mobile

## Technology Stack

- **Backend**: Flask, SQLAlchemy, MySQL
- **Frontend**: Bootstrap 5, Chart.js, JavaScript
- **ML**: Scikit-learn, XGBoost, Pandas
- **Additional**: ReportLab, Flask-Login, Flask-WTF

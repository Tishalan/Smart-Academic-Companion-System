Baseline Model
Logistic Regression

Logistic Regression serves as our baseline. It is a linear model that outputs probabilities using a sigmoid function. It excels when features have a linear relationship with the target logging simple, interpretable decision boundaries.

Ensemble Model
Random Forest

Random Forest is an ensemble of decision trees trained via the bagging method. By averaging predictions from multiple uncorrelated trees, it significantly reduces overfitting and captures complex non-linear patterns in student behavior.

Boosting Model
XGBoost (Selected)

eXtreme Gradient Boosting (XGBoost) utilizes gradient boosting framework. It builds trees sequentially, where each new tree corrects errors from previous ones. It includes L1/L2 regularization to prevent overfitting.

5.4.4 Evaluation Metrics
Accuracy

Overall correctness. Used for high-level selection.

Precision

Correct positive ratio. Used to avoid false alarms.

Recall

Sensitivity. Used to find all at-risk students.

F1-Score

Harmonic balance for imbalanced data.

ROC-AUC

Class distinction ability. Measures probability calibration.

Confusion Matrix

Visualizes FP/FN for error analysis.
